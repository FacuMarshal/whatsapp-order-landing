# WhatsApp Order Landing

Modern product ordering landing page with automated WhatsApp checkout flow.

---

## Overview

This project was created for a meat distribution business to simplify product ordering through mobile devices.

Users can:

- Browse products
- Add items to cart
- Generate automatic order summaries
- Send orders directly through WhatsApp

The platform was designed for fast mobile ordering and B2B distribution workflows.

---

## Features

- Responsive landing page
- Product catalog system
- Dynamic cart logic
- Automatic WhatsApp message generation
- Mobile-first design
- Business ordering workflow
- Fast checkout experience

---

## Tech Stack

- HTML
- CSS
- JavaScript
- WhatsApp API links
- Responsive UI

---

## Workflow

User selects products → Cart generates order → WhatsApp opens with pre-filled message
